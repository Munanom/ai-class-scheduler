USIU LAB Scheduler

Welcome to the Class Lab Assignment Program

This program allows you to upload an Excel file containing class schedule information and automatically assigns classes to various labs based on their timing.

Uploading Your Excel File

To get started, please follow these steps:

  1. Prepare your class schedule data in an Excel file with the following format:
  • Column 1: Start Time (e.g., 07:00)
  • Column 2: Days (e.g., TR for Tuesday and Thursday)
  • Column 3: Course Number (e.g., APT1040)
  • Column 4: Section (e.g., C)
  • Column 5: Course Title (e.g., INTRODUCTION TO WEB DESIGN & APPLICATIONS)
  • Column 6: Faculty (e.g., Staff)
  • Column 7: Capacity (e.g., 35)
  1. Save your Excel file.
  2. Click on the "Choose File" button below and select your Excel file.
  3. Click the "Upload" button to submit your file.

Class Lab Assignment Process

Once you've uploaded your file, the program will process the data and assign classes to the available labs according to their timing. Lab capacities are not considered in this assignment process.

Viewing the Results

After the assignment process is complete, you'll be presented with the lab assignments for each time slot. Each assignment includes the class title, course number, section, professor, and the assigned lab.
